Definition · 定义
03 / 20
一句话理解
保存提示词告诉它“怎么开始”。 Skill 告诉它“怎样交付一份合格工作”。

它包含触发条件、流程步骤、好输出样例、失败处理、工具要求和交付格式。

Saved prompt is a start; Skill is a trained employee

Filesystem · 它住在哪里
06 / 20
没有神秘基础设施

Skill 本质上，
就是一个文件夹。

每个 Skill 文件夹里有一个 `SKILL.md`。Claude Code 可以放在项目级 `.claude/skills/`，也可以放在全局 `~/.claude/skills/`。

低门槛才是重点：一个目录，一个说明文件，一套可重复执行的工作标准。
Folder + SKILL.md

Activation · 触发描述
11 / 20
description 要“激进而具体”

告诉 Claude：
什么时候该用，什么时候不该用。

描述里要列出触发短语、适用场景和排除场景。这样 Claude 才能在正确时机自动调用它，而不是每次等你提醒。

坏描述：处理邮件。
好描述：给参加过 webinar 的潜在客户写跟进邮件，引用其参加场次、一个案例和 15 分钟 demo CTA。
Specific triggers beat vague labels
Frontmatter · name in kebab-case, description with trigger phrases

60 天后的差别
多数人会继续每天输入同一段话。 建立 Skill 库的人， 会运行一套完全不同的操作系统。
